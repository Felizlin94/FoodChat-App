# FoodChat-App

## 使用方法
1.打開終端機Terminal；

2.輸入指令以下載本專案：

 `git clone https://github.com/Felizlin94/FoodChat-App.git`
 
3.進入專案資料夾：

`cd FoodChat-App`

4.安裝npm：

`npm install`

5.啟動npm

`npm start`

6.在您的瀏覽器輸入以下網址以瀏覽本專案：

##### `http://localhost:3000`

----------------------------------------

開啟server，以享受websocket即時傳輸：

1.打開另一個終端機Terminal；

2.進入專案資料夾：

`cd FoodChat-App`

3. 進入server所在位置：

`cd src`

4. 開啟server：

`node server`

此時應用程式可以順利運作
